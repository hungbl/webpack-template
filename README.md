# Webpack 5 Template

This template contains basic feature about Webpack 5, Babel, ESlint... for my personal purpose only.

## Installation

Use the package manager npm to install all dependencies contains on package.json

```bash
npm install
```

## Usage

```bash
# run dev
npm run start

# build prod
npm run build
```
